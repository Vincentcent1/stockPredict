# Stock Prediction with News Input
Code base for stock prediction project for Computational Data Science SUTD ISTD Term 6

## Scrapper Support
* [MontlyFool](fool.com)
* [Investopedia](investopedia.com)
* [Reuters](reuters.com)
* [CNBC](cnbc.com)
* [MarketWatchNewsViewer](marketwatch.com)
